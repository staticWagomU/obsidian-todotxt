# コンテンツ作成LLM 運用保守プラン

**月額 100,000円（税別）**

---

## 1. プラン概要

本プランは、コンテンツ作成LLMシステムの安定稼働を目的とした運用保守サービスです。

システムの停止や不具合発生時に、**一次復旧（暫定回避）まで**を対応範囲とし、継続的な改善や開発対応は含まないことで、無理のない・継続可能な運用を前提としています。

---

## 2. 提供内容

### 2.1 監視・死活確認

**対象システム：**
- Cloud Run / Cloud Run Jobs
- データベース
- 外部LLM API 等

**監視内容：**
- ジョブ失敗・エラー率の確認
- DB接続エラーの確認
- 外部LLM API失敗率の確認

**監視方法：**
- Cloud Monitoring によるアラート監視
- 監視頻度：5分間隔での死活確認
- アラート条件：エラー率5%以上、応答時間30秒以上（※要調整）

---

### 2.2 障害一次対応（復旧オペレーション）

**対応時間上限：** 月2時間まで

**対応範囲：**
- ジョブ再実行・リトライ対応
- 設定差し戻し対応（直近変更の巻き戻し）
- 軽微なスケール・制限調整（同時実行数／タイムアウト／メモリ等を元の状態へ戻す対応）
- 暫定回避対応（特定入力の一時停止、機能フラグによる停止など）
- 障害一次切り分け（アプリ／DB／外部LLM／権限／ネットワークの切り分け）
- 障害状況の共有・報告（原因推定・次アクション・復旧見込みは推定として共有）

**対象外：**
- 原因究明の深掘り
- 恒久対応
- コード修正

**エスカレーション対応：**

一次復旧で解決しない場合、以下のフローで対応します：

1. 状況報告書を作成（対応時間内）
2. 恒久対応の見積もりを別途ご提示
3. お客様承認後、個別契約として対応

---

### 2.3 問い合わせ対応

**対応内容：**
- 「動かない」「結果が出ない」等に対する一次回答
- ログ確認および状況整理
- 切り分け結果の共有

**上限：**
- 月3件まで
- 合計60分まで
- テキスト対応のみ（メール／チャット）

---

### 2.4 コストチェック

**対応内容：**
- LLM利用費・インフラ費の前月比チェック
- 急増（前月比150%以上等）が見られた場合の通知

## 5. 障害と品質問題の定義

### 5.1 障害（本プラン対応範囲）

以下の状態を「障害」として対応対象とします：

- システムがHTTPエラー（5xx）を返す
- ジョブが異常終了する
- 応答が一切返らない（タイムアウト）
- データベースに接続できない
- 認証・認可エラーでアクセスできない

### 5.2 品質問題（本プラン対象外）

システムは正常に動作しているが、出力内容に関する問題：

- 出力内容が期待と異なる
- 根拠が弱い、参照情報が不足している
- 文体・トーンが合わない
- 精度が落ちたと感じる
- 回答内容にばらつきがある
- 応答速度の改善要望

※ 品質問題への対応は「品質改善パック」（後述）にて承ります

## 6. 責任範囲

### 6.1 対応範囲

本プランは、お客様システムの運用保守を対象とします。

### 6.2 対応範囲外

以下は本プランの対応範囲外となります：

| 原因 | 対応 |
|------|------|
| クラウドプロバイダー（GCP等）起因の障害 | 切り分け・状況共有まで。復旧はGCP依存 |
| 外部LLM APIプロバイダー（OpenAI/Anthropic等）起因の障害 | 切り分け・状況共有まで。復旧は外部依存 |
| お客様による設定変更・操作に起因する障害 | 対応可能だが、対応時間としてカウント |
| お客様環境固有のネットワーク障害 | 切り分けまで。復旧はお客様側で対応 |

## 7. 免責事項・損害賠償

### 7.1 免責事項

- 本サービスは障害の完全な防止・復旧を保証するものではありません
- 暫定対応による一時的な機能制限が発生する場合があります
- 外部サービス（GCP、LLM API等）の障害・仕様変更による影響は保証対象外です

### 7.2 損害賠償の範囲

- 間接損害、逸失利益、データ消失による損害は補償対象外とします
- 本サービスに起因する損害賠償の上限は、**直近3ヶ月分の月額料金**を上限とします
- お客様の顧客に対する損害について、弊社は責任を負いません
