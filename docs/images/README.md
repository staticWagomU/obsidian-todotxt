以下のスクリーンショットを後日追加する予定です：

1. **priority-badge.png** - 優先度バッジ表示のスクリーンショット
2. **due-display.png** - 期日表示と期日超過ハイライトのスクリーンショット
3. **threshold-grayout.png** - 開始日によるグレーアウト表示のスクリーンショット
4. **internal-links.png** - Obsidian内部リンク対応のスクリーンショット
5. **external-links.png** - 外部リンク表示のスクリーンショット
6. **form-input.png** - 構造化フォーム入力UIのスクリーンショット
7. **settings-screen.png** - 設定画面のスクリーンショット
